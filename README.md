## Instructions on running project locally:

Clone the project

```bash
  git clone https://github.com/prathikshetty2002/Codeissance_41_Devshaks.git
```

Install dependencies:

```bash
  cd www.shield.com/
  
  yarn install
```

Run Project on terminal

```bash
  yarn dev
```

Start the server

```bash
  http://localhost:3000
```
